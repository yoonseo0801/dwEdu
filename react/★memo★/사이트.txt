Apache가 하는 일
        역할	                        설명
🌍 웹페이지 제공	사용자가 요청한 페이지(HTML, CSS, JS 등)를 전송
⚙️ 요청 처리	브라우저의 GET/POST 요청을 받아서 응답 생성
🔒 보안 관리	HTTPS(SSL 인증서) 설정, 접근 제한
🧩 확장 기능	모듈을 추가해 PHP, Python 등 다양한 언어 실행 가능
🧾 로그 관리	접속 기록, 에러 로그 저장
